第二次讨论会——主题：博客搭建

DIY - 02

如何实现？

内容二

  1. Markdown语法：一种简便的记笔记的语法，旨在快速编写简洁美观的文章。
    支持markdown的软件会让你的markdown语法写的文章自动添加一些格式
  2. 图床：一种专门存图片的技术，可以将图片存入互连网，然后将其变成可访问的链接的方式。
    主要是来配合markdown语法编辑器，因为markdown不支持存图片，所以可以通过使用图床的链接来访问的形式。
    同时还可以缓解网站服务器压力
  3. DNS域名解析: 可以将我们的网站 URL 更改为我们自己的网址
  4. CDN加速：源站的资源缓存到位于全国各地的CDN节点上，用户请求资源时，就近返回节点上缓存的资源，而不需要每个用户的请求都回您的源站获取，避免网络拥塞、分担源站压力，保证用户访问资源的速度和体验。

主要是进一步优化，加一些稍微难理解的一部分内容

DNS域名解析

原理

我们在互联网上通过 IP地址 访问服务器，比如京东、淘宝、百度、bilibili等，但 IP 地址是一串字面上没有任何意义的数字（如果你感觉不出来可以试试 IPv6 的地址），为了让方便人类的记忆和书写 ，提出了用 域名 代替 IP地址 的方案。

当写一个域名的时候，实际上电脑会去对应的 DNS服务器里，查询域名对应的 IP地址，返回给你电脑，然后你的电脑就回去访问这个 IP地址。

这些 DNS服务器 掌握在那些互联网大厂中，比如 阿里、腾讯、华为等。

如何拥有一个自己的 域名

域名（Domain）

顶级域名（.com .cn .top .eu .xyz .我爱你 .中国）

二级域名（.com.cn .edu.cn）

虽然这些域名看起来有区别，价格也不一样，其实……没啥区别………………
唯一需要主义的就是一部分国外的域名在中国不支持实名认证。没有通过实名认证的域名无法使用很多国内服务，只能用海外的服务。不过大部分都是支持的。

解析内容介绍

这里我先重新介绍一下我们的域名，我们买的是只是域名

比如 www.baidu.com，我们买的只是baidu.com，至于前面的 www ，是我们在这个位置自己加上去的。所以我们其实一个域名可以有很多很多个 URL，比如我这里可以加 test 就会有 test.baidu.com，加上 blog 就会有 blog.baidu.com，甚至可以加上几级，比如加上hi.hello 就会有 hi.hello.baidu.com ，哪怕是加上 one.two.three.four 就会有 one.two.three.four.baidu.com 。

再回到上图，

  • 其实 Name 项就是 网址前缀，最前面那一部分。
  • Type项 就是解析的内容，A 是解析一个 IPv4 地址，AAAA 是解析一个 IPv6 地址，CNAME 是解析另一个网址，把网址A解析成网址B
  • Content，就是你要解析的内容，这个部分看你 Type 项选择

CDN加速

原理

什么是CDN？它解决了什么难题？5分钟让你明明白白！-腾讯云开发者社区-腾讯云 (tencent.com)

CDN英文全称Content Delivery Network，中文翻译即为内容分发网络。它是建立并覆盖在承载网之上，由分布在不同区域的边缘节点服务器群组成的分布式网络。

CDN应用广泛，支持多种行业、多种场景内容加速，例如：图片小文件、大文件下载、视音频点播、直播流媒体、全站加速、安全加速等。
